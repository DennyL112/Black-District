## 📌 Übersicht

Dieses Repository dient als zentrale Entwicklungsbasis für den **Black District** Roleplay Server.

Enthalten ist der serverrelevante **`resources/`**-Ordner mit allen Entwicklungs-, Verwaltungs- und Integrationsbereichen.

## 📦 Inhalte

* **Scripts** — Gameplay-Logik, Systeme und Funktionen
* **Fahrzeuge** — Addon-Fahrzeuge, Handling und Fahrzeugbearbeitung
* **Maps / MLOs** — Gebäude, Interiors und Map-Erweiterungen
* **Bilder** — Grafiken, Icons und sonstige Bilddateien

## 🔁 Entwicklungsablauf

```text
Developer → Prüfung → Integration → Live-Server
```

1. Entwicklung im eigenen Ordner unter `Developer/`
2. Inhalte lokal erstellen und testen
3. Fertige Inhalte im eigenen Developer-Ordner ablegen
4. Development-Leitung informieren
5. Prüfung und Integration in den Server

## 🔒 Regeln

### Developer-Bereich

* Es wird ausschließlich im eigenen Developer-Ordner gearbeitet.
* Änderungen an anderen Developer-Ordnern sind nicht erlaubt.
* Fertige Inhalte werden sauber im eigenen Bereich abgelegt.

### BD-Server

* Direkte Änderungen am `BD-Server/`-Bereich sind nicht erlaubt.
* Dateien dürfen nicht eigenständig überschrieben oder ersetzt werden.
* Zugriff, Prüfung und Integration erfolgen ausschließlich über die Development-Leitung.
